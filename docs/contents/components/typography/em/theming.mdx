## Theming

The `Em` is a [single part component](/styled-system/theming/component-styles#single-part-components).

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const Em: ComponentStyle<"Em"> = {
  baseStyle: {
    fontStyle: "italic",
  },
}
```
