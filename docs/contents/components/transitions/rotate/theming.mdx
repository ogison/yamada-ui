## Theming

The `Rotate` is a [single part component](/styled-system/theming/component-styles#single-part-components).

:::note
`Rotate` inherits the style from [Airy](/components/transitions/airy).
:::

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::

```ts
export const Rotate: ComponentStyle<"Rotate"> = mergeStyle(Airy)()
```
