`title`と`description`が設定されていない場合は、`Step`コンポーネントに`aria-label`を設定します。

```ts
const steps: Steps = [
  { "aria-label": "シーズン 1" },
  { "aria-label": "シーズン 2" },
  { "aria-label": "シーズン 3" },
]
```

## ARIAロールと属性

| コンポーネント  | ロールと属性         | 使い方                                       |
| --------------- | -------------------- | -------------------------------------------- |
| `StepSeparator` | `role='separator'`   | セパレータであることを示します。             |
| `StepStatus`    | `aria-hidden='true'` | 要素をアクセシビリティツリーから除外します。 |
